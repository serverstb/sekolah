# **App Name**: ScanAttend

## Core Features:

- Barcode Scanning: Scan student barcodes to record attendance.
- Attendance Recording: Record the date and time of each student's attendance.
- Attendance Reporting: Generate attendance reports for each student or class.
- User Authentication: Secure user authentication for admins.

## Style Guidelines:

- Primary color: Dark blue (#2E4A62) for a professional and trustworthy feel.
- Background color: Light gray (#F0F4F7) to provide a clean and neutral backdrop.
- Accent color: Teal (#33A3A3) for highlighting important actions and information.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, will be used throughout the application to provide a clean and modern look.
- Use simple, clear icons to represent actions and data, ensuring ease of understanding and quick navigation.
- Design a clean and intuitive layout with clear hierarchy, focusing on ease of use for both admins and students.
- Subtle animations, such as fade-in effects, can be used to enhance the user experience without being distracting.
